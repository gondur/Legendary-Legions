(In Engish)
To play Legendary Legions by net one of the players must be server and the other must be client.

SERVER settings:
(Config your firewall to forward 9001 port)
- Select: Options->Multiplayer->(first button)Server
- Choose side, which you will play (Server player has right to choose and starts the game as first)
- Go back to Main Menu and select map and start new game or load saved game

CLIENT settings:
- Select: Options->Multiplayer->(first button)Server
- Enter IP address
- Go back to Main Menu and start new game

CHAT:
Just type text and press ENTER to send it.
